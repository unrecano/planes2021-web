Planes 2021

Planes 2021 es un proyecto creado para revisar y analizar planes de gobierno de los candidatos para las próximas elecciones.

Este proyecto se crea como un ejercicio que hace uso de técnicas como WEB SCRAPING y Natural Language Processing

Los datos son extraídos de las páginas del Jurado Nacional de Elecciones y son procesados con Python.

El código del scraping y el de este sitio están en Gitlab y Github.
